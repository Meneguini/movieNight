## What is Movie Night?

Movie Night is web application created using Django, postgreSQL, HTML, CSS and JavaScript.
You can access the web app at https://my-movie-night.herokuapp.com/

## Technologies

- Django
- PostgreSQL
- JavaScript
- Infinite Scroll
- The Movie Database API
- Youtube embedded link
- User authentication
